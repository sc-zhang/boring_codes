## Agent-Based Modeling
```bash
usage: python3 ABM.py [-h] [-s SATISFACTION] [-d DENSITY] [-e ELEMENT] [-a AREA]
              [-i ITERATION]

options:
  -h, --help            show this help message and exit
  -s SATISFACTION, --satisfaction SATISFACTION
                        ratio of satifaction, should >0 and <1, default=0.3
  -d DENSITY, --density DENSITY
                        density of element, should >0 and <1, default=0.5
  -e ELEMENT, --element ELEMENT
                        types of elements, default=3
  -a AREA, --area AREA  area**2=area size, default=10
  -i ITERATION, --iteration ITERATION
                        iteration cycles, default=1e6
```
